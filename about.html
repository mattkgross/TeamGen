General Info


This tool was created in order to provide a simple means to register players and create teams of an evenly distributed skill level. Please let me know if there's any trouble you run into.

Please Note: This is not a distributed system. Each browser will have its own data. Team data cannot be shared across multiple browsers or computers. One tournament director, one computer, one player list.


Adding Players


To add players to the list, simply input their information (first & last names and rating required). The player list is kept even if the browser is closed. So, if you are starting a new tourney, be sure to clear out the previous team. Players can be individually deleted from the list by accessing "View Player List" and clicking the "×" by their name.


Generating Teams


Once you have input all of the players, simply select the number of teams you would like to create. You also have the option of downloading all of the player information to a CSV file for future reference. Don't like the teams? Simply regenerate. Again, once you're done, remeber to clear the player list for the next use. The player list will persist until you delete it.
