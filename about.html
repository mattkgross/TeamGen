General Info


This tool was created in order to provide a simple means to register players and create teams of an evenly distributed skill level. Please let me know if there's any trouble you run into.

Please Note: This is not a distributed system. Each browser will have its own data. Team data cannot be shared across multiple browsers or computers. One tournament director, one computer, one player list.


Adding Players


To add players to the list, simply input their information (first & last names, and rating required). The player info is kept even if the browser is closed. So, if you are starting a new tourney, be sure to clear out the current team.
